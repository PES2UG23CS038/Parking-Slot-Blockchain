# Blockchain-Based Parking Slot Allocation System

## Overview
This project implements a decentralized parking slot allocation system using blockchain technology. It ensures transparency, prevents double-booking, and maintains an immutable record of all transactions.

## Features
- Add parking slots
- Reserve slots with time duration
- Release slots
- View available slots
- Maintain reservation history

## Technologies Used
- Solidity
- Remix IDE
- Ethereum (Simulation)

## Smart Contract
Located in `ParkingSystem.sol`

## How to Run
1. Open Remix IDE
2. Paste the smart contract
3. Compile and deploy
4. Interact with functions
